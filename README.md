# EE2010 Pub POS group 6

This repository contains the code for the EE2010 Pub Point Of Sale (POS) system developed by group 6. 
The system is designed to streamline the ordering and payment process in a pub environment.

## Features
- User-friendly interface for order taking
- Bill generation and display
- Item management with categories and pricing
- Management Real-time dashboard for stock tracking

## Technical Details
- Uses polymorphic associations to manage relationships between bills, items.
- Focuses on modular and maintainable code structure.
